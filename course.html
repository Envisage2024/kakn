Practical Computer Applications and ICT Integration

For Teachers and Adults

Course Description

This course equips teachers, adults, and professionals with practical computer application skills and introduces them to ICT integration in classrooms and workplaces. It covers basic digital literacy and builds up to real-world ICT use cases.

Target Audience

  • Teachers (primary, secondary, tertiary)
  • Adult learners with little or no ICT background
  • NGO staff, church leaders, community trainers
  • Civil servants and office workers
  • Parents supporting children's digital learning

Overall Learning Outcomes (OLOs)

By the end of the course, learners will be able to:

  • Confidently operate a computer and manage files/folders
  • Create professional documents, spreadsheets, and presentations
  • Use email and internet tools safely and effectively
  • Apply ICT in teaching, lesson planning, and assessment
  • Integrate ICTs into classroom and community activities
  • Practice safe, ethical, and responsible ICT use

Course Modules

7 comprehensive modules covering essential ICT skills

Module 1: Introduction to Computers & Digital Literacy

ICT basics, hardware and software fundamentals, operating a computer, and file management. Learn to create, organize, and backup files safely.

  • ICT basics: importance and applications
  • Hardware: input, output, storage devices
  • Software: system, application, utility programs
  • File management: creating, renaming, saving, backups
  • Practical: Power on/off safely, create/save folders, backup to flash/cloud

Module 2: Word Processing for Documentation

Master document creation with MS Word or Writer. Format text, create tables, insert images, and use professional tools.

  • Interface of MS Word/Writer
  • Formatting text and paragraphs
  • Page layout: margins, headers/footers, numbering
  • Tables, images, SmartArt
  • Tools: spell check, track changes
  • Practical: Draft lesson plan, school circular, student report with tables

Module 3: Spreadsheets for Data Management

Learn Excel basics: formulas, functions, charts, and data analysis for school and office work.

  • Worksheet structure: rows, columns, cells
  • Formatting cells and ranges
  • Formulas/functions: SUM, AVERAGE, MIN, MAX
  • Sorting, filtering, autofill
  • Charts: bar, pie, line
  • Practical: Class register with averages, performance chart, budget sheet

Module 4: PowerPoint for Presentations & Lessons

Create engaging presentations with professional design, multimedia, and animations.

  • Slide design: layouts, themes, backgrounds
  • Inserting text, images, multimedia
  • Transitions and animations
  • Presenter tools and handouts
  • Best practices in educational slides
  • Practical: Create 5-slide lesson, add animations, deliver presentation

Module 5: Email & Online Communication Tools

Professional email communication, online meetings, and collaborative tools for modern workplaces.

  • Creating/managing email accounts
  • Writing professional emails with attachments
  • Inbox organization and etiquette
  • Online meetings: Zoom/Meet basics
  • Collaborative tools: Google Docs, Drive
  • Practical: Compose/send email, schedule Zoom, co-edit document

Module 6: Digital Classroom Tools & ICT Integration

Explore ICT's role in Uganda's CBC and learn to use digital classroom tools effectively.

  • Role of ICT in CBC teaching
  • Tools: Google Classroom, Moodle, projectors
  • Blended learning strategies
  • ICT in subjects: science, math, humanities, languages
  • Practical: Upload assignment on Classroom, plan ICT-supported lesson

Module 7: ICT Safety, Security, and Ethics

Protect yourself online with cybersecurity best practices and ethical ICT use.

  • Passwords, two-factor authentication
  • Recognizing scams and phishing
  • Safe browsing and antivirus use
  • Privacy and data protection
  • Ethics: plagiarism, online conduct
  • Practical: Digital safety checklist, analyze case study, ethics discussion
